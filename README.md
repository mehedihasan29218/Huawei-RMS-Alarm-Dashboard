# Huawei-RMS-Alarm-Dashboard
Overview
This project delivers an interactive Power BI dashboard for analyzing telecom tower alarms generated by the Remote Monitoring System (RMS). Alarm data is automatically downloaded from the RMS platform using RPA, transformed through Power Query, and analyzed using DAX measures and calculated columns to generate business insights.

Business Use Case
•	Centralized monitoring of tower alarms
•	Faster identification of recurring and critical issues
•	Improved operational visibility for NOC and management
•	Data-driven decision-making using alarm trends

End-to-End Workflow
1.	RPA downloads alarm data from Huawei RMS platform
2.	Data ingested into Power BI
3.	Power Query performs ETL and data cleansing
4.	DAX measures and calculated columns generate KPIs
5.	Interactive dashboard delivers operational and management insights

Tools & Technologies
•	RPA (UiPath) – Automated data extraction
•	Power BI – Data modeling and visualization
•	Power Query – ETL and data transformation
•	DAX – Measures, calculated columns, KPIs

Key Insights
•	Alarm frequency by site and region
•	Critical vs non-critical alarm distribution
•	Repeated alarm trend analysis
•	Performance visibility across time

Dashboard Features
•	Interactive filtering and drill-down
•	Time-based alarm analysis
•	Clean, management-ready visuals
